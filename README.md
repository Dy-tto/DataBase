# SELECT
## SQL문 작성 규칙
1. 대소문자를 구별하지 않는다.
2. 한 줄 또는 여러 줄로 작성할 수 있다.
3. 들여쓰기로 가독성을 높인다.
4. 명령어를 대문자로 작성하고 나머지를 소문자로 작성해서 가독성을 높인다.
